## 🌮 taco-tui 🌮

A Taco Bell order simulator ran in your terminal, expanding on what I
learned from the [burrito index program](https://github.com/colinkautz/burrito-index) I wrote.

### Requirements
- Go 1.26+
- A love for Taco Bell

### Instructions
1. Clone this repo
2. Run `go build`
3. Run `./taco-tui` (or `taco-tui.exe` on Windows)
4. Follow the prompts (enter a zip, select a store, add items to cart, etc.)
